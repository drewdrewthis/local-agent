# Agent Notes Directory

Categorized knowledge base for efficient assistant operation. Reference these files instead of searching conversation history.

## File Structure

- **`storysnacks.md`** - Storysnacks project status, preferred formats, tracked issues
- **`budget-tracking.md`** - Financial tracking preferences, Google Sheets rules, business context
- **`email-calendar.md`** - Email/calendar management, agent identity, communication guidelines
- **`user-preferences.md`** - Response formats, work patterns, communication style preferences

## Usage Guidelines

1. **Reference First**: Check relevant note files before asking user for clarification
2. **Update Regularly**: Add new preferences/patterns as they emerge
3. **Ask When Uncertain**: If context unclear, ask user directly rather than exhaustive history searches
4. **Categorize New Topics**: Create new files for new subject areas as needed

## Maintenance

- Keep files concise and focused
- Update timestamps and status when information changes
- Use consistent formatting across files
- Commit changes with conventional commit messages
